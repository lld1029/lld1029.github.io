配置 在config.toml文件中，在中定义以下变量params：

author：作者姓名 description：作者简介 avatar：包含作者头像图像的文件的路径 menu_item_separator：每个菜单项之间的分隔符。允许HTML（默认：“-”） favicon：您的favicon.ico文件的绝对路径（默认值：“ / favicon.ico”） 要添加菜单项，请在中添加以下行menu：

[[menu.main]] identifier = “tags” name = “Tags” url = “/tags/”

要添加社交链接，请在中添加以下行params：

[[params.social]] name = “github” url = “https://github.com/vaga" 要更改主题颜色，请在中添加以下行params：

[params.style] darkestColor = “#d35050” darkColor = “#212121” lightColor = “#f5e3e0” lightestColor = “#f5f5f5” primaryColor = “#fff” 如果需要以上主题颜色，则可以查看exampleSite / config.toml文件。

造型 要使用SCSS覆盖样式，添加一个名为_extra.scss来{your site directory}/assets/css/

注意：Hugo版本有两个版本，hugo和hugo_extended。您需要hugo_extended自动编译您的scss。
